# M3a.1 — AWS EC2 + RDS Shards (on-demand) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship two production-shaped AWS shards — `aws-ec2` and `aws-rds`, on-demand only — end-to-end: DuckDB-based Python ingest from AWS Pricing API JSON (fixtures used in tests, same code path runs live in CI later), SQLite packaging via the shared packager, pure-Go catalog lookup, and four new Cobra leaves (`sku aws ec2 {price,list}`, `sku aws rds {price,list}`) that return valid §4 envelopes. The M1 inline updater is extended to accept both shard names; the full `internal/updater` module and daily CI pipeline remain deferred to m3a.3.

**Architecture:**

1. **Pipeline half (`pipeline/ingest/aws_ec2.py`, `pipeline/ingest/aws_rds.py`):** stream an AWS Pricing API offer JSON (either from a fixture under `pipeline/testdata/` or the live `https://pricing.us-east-1.amazonaws.com/offers/...` URL), and use **DuckDB** as the normalization engine. DuckDB's `read_json` walks the nested `products` + `terms.OnDemand` shape in one SQL pass, joins per-region product rows with their on-demand term rows, filters to the supported enum surface (linux/windows/rhel OS on EC2; mysql/postgres/mariadb engines × single-az/multi-az on RDS), computes `terms_hash` via a Python UDF bound to `pipeline.normalize.terms.terms_hash`, canonicalizes region via `pipeline/normalize/regions.yaml`, and emits NDJSON rows to stdout (same shape the OpenRouter path emits).
2. **Packager (`pipeline/package/build_shard.py` — unchanged binary, touched only if a new `resource_attrs.extra` JSON field is needed):** NDJSON rows → SQLite shard with the full §5 schema + metadata.
3. **Client half (pure Go):** `internal/catalog.LookupVM` and `internal/catalog.LookupDBRelational` execute point lookups + list scans bound by the M1 index set (`idx_skus_lookup` leading with `resource_name`). `cmd/sku/aws.go` registers the `aws` subtree; `cmd/sku/aws_ec2.go` + `cmd/sku/aws_rds.go` wire the four leaves through the existing M2 `output.Pipeline` so presets/jq/fields/format all work out of the box. `cmd/sku/update.go` is generalized to take a per-shard URL map so `sku update aws-ec2` and `sku update aws-rds` both work against the bootstrap release assets.

**Tech Stack:** Python 3.11+, `duckdb>=0.10` (new dep), existing `pipeline/package/build_shard.py` (unchanged), Go 1.25, `modernc.org/sqlite`, `github.com/spf13/cobra`. No new Go deps.

**Spec ref:** `docs/superpowers/specs/2026-04-18-sku-design.md` §3 (format stack, shard granularity, update flow), §4 (command tree, global flags, output schema), §5 (kind taxonomy, terms, region normalization, indexes), §6 (unit/integration test targets), §9 M3a entry.

**Explicit non-goals for m3a.1** (declared up front to prevent scope creep into sibling sub-plans):
- `aws-s3`, `aws-lambda`, `aws-ebs` shards → m3a.2.
- `aws-dynamodb`, `aws-cloudfront` shards → m3a.3.
- `aws-ec2-commitments` / `aws-rds-commitments` (reserved / savings-plan) → deferred past M3a per spec §3.
- Raw sibling shard (`aws-ec2-raw.db.zst`) — no raw ingest until m3a.3 when the full updater module lands.
- `internal/updater` extraction with delta chain, manifest walking, ETag — m3a.3. `sku update` stays inline, one-shot baseline per M1 pattern.
- `data-daily.yml` CI workflow — m3a.3.

**Out-of-scope for m3a.1 within each shard:**

"""Shared AWS ingest helpers: region normalization, SKU ID passthrough."""

from __future__ import annotations

from dataclasses import dataclass
from pathlib import Path
from typing import Mapping

import yaml

_REGIONS_YAML = Path(__file__).resolve().parent.parent / "normalize" / "regions.yaml"


@dataclass(frozen=True)
class RegionNormalizer:
    """Maps (provider, provider-region) -> canonical group."""

    table: Mapping[tuple[str, str], str]

    def normalize(self, provider: str, region: str) -> str:
        key = (provider, region)
        try:
            return self.table[key]
        except KeyError as exc:
            raise KeyError(f"{provider}/{region}") from exc


def load_region_normalizer() -> RegionNormalizer:
    """Load the repo's regions.yaml and build a (provider, region) -> group map."""
    with _REGIONS_YAML.open() as fh:
        doc = yaml.safe_load(fh)
    table: dict[tuple[str, str], str] = {}
    for group, entries in (doc.get("groups") or {}).items():
        for entry in entries:
            key = (entry["provider"], entry["region"])
            table[key] = group
    return RegionNormalizer(table)
```

"""Tiny wrapper around DuckDB: open an in-memory conn with terms_hash bound as a UDF."""

from __future__ import annotations

import json
from typing import Any

import duckdb

from normalize.terms import terms_hash


def _terms_hash_udf(
    commitment: str, tenancy: str, os: str,
    support_tier: str, upfront: str, payment_option: str,
) -> str:
    return terms_hash({
        "commitment": commitment or "",
        "tenancy": tenancy or "",
        "os": os or "",
        "support_tier": support_tier or "",
        "upfront": upfront or "",
        "payment_option": payment_option or "",
    })


def open_conn() -> duckdb.DuckDBPyConnection:
    """Return an in-memory conn with `sku_terms_hash(...)` bound as a scalar UDF."""
    con = duckdb.connect(":memory:")
    con.create_function(
        "sku_terms_hash",
        _terms_hash_udf,
        parameters=["VARCHAR"] * 6,
        return_type="VARCHAR",
    )
    return con


def dumps(obj: Any) -> str:
    """Compact JSON — identical to openrouter.py's encoding."""
    return json.dumps(obj, separators=(",", ":"), ensure_ascii=False)
```

"""Normalize AWS EC2 offer JSON into sku row dicts via DuckDB.

Spec §3 (format stack): DuckDB read_json streams the nested offer file;
one SQL query joins products + OnDemand terms + priceDimensions into
one row per (sku_id, pd_sku) and we emit NDJSON.
"""

from __future__ import annotations

import argparse
import sys
from pathlib import Path
from typing import Any, Iterable

from normalize.enums import apply_kind_defaults, load_enums
from normalize.terms import terms_hash

from ._duckdb import dumps, open_conn
from .aws_common import load_region_normalizer

_PROVIDER = "aws"
_SERVICE = "ec2"
_KIND = "compute.vm"

# Keep the OS mapping narrow — only variants we ship in v1.
_OS_MAP: dict[str, str] = {"Linux": "linux", "Windows": "windows", "RHEL": "rhel"}
# Tenancy comes through as Shared / Dedicated / Host; we drop Host for m3a.1.
_TENANCY_MAP: dict[str, str] = {"Shared": "shared", "Dedicated": "dedicated"}

_NORMALIZE_SQL = """
WITH ondemand AS (
  SELECT sku_id, pd.unnest.key AS pd_key, pd.unnest.value AS pd
  FROM (
    SELECT key AS sku_id, unnest(map_entries(value))
    FROM (SELECT unnest(map_entries(terms->'OnDemand')) FROM offer),
  )
),
pd_flat AS (

    enums = load_enums()
    normalizer = load_region_normalizer()

"""Normalize AWS RDS offer JSON into sku row dicts via DuckDB.

Spec §5 db.relational kind. Engine and deployment-option ride the
terms.tenancy and terms.os slots — see enums.yaml comment for rationale.
"""

from __future__ import annotations

import argparse
import sys
from pathlib import Path
from typing import Any, Iterable

from normalize.enums import apply_kind_defaults, load_enums
from normalize.terms import terms_hash

from ._duckdb import dumps, open_conn
from .aws_common import load_region_normalizer

_PROVIDER = "aws"
_SERVICE = "rds"
_KIND = "db.relational"

_ENGINE_MAP = {"PostgreSQL": "postgres", "MySQL": "mysql", "MariaDB": "mariadb"}
_DEPL_MAP = {"Single-AZ": "single-az", "Multi-AZ": "multi-az"}

_SQL = """
WITH ondemand AS (
  SELECT sku_id, pd.unnest.key AS pd_key, pd.unnest.value AS pd
  FROM (
    SELECT key AS sku_id, unnest(map_entries(value))
    FROM (SELECT unnest(map_entries(terms->'OnDemand')) FROM offer),
  )
),
pd_flat AS (
  SELECT sku_id, pd->>'unit' AS unit, CAST(pd->'pricePerUnit'->>'USD' AS DOUBLE) AS usd
  FROM ondemand
)
SELECT
  p.sku AS sku_id,
  p.attributes->>'instanceType' AS instance_type,
  p.attributes->>'regionCode' AS region,
  p.attributes->>'databaseEngine' AS engine_raw,
  p.attributes->>'deploymentOption' AS depl_raw,
  p.attributes->>'licenseModel' AS license_model,
  p.attributes->>'vcpu' AS vcpu_raw,
  p.attributes->>'memory' AS memory_raw,
  f.usd, f.unit
FROM (SELECT unnest(map_entries(products)).value AS p FROM offer) pr
JOIN pd_flat f ON f.sku_id = pr.p.sku
WHERE pr.p.productFamily = 'Database Instance'
"""


def _parse_memory(raw: str) -> float:
    return float(raw.split()[0])


def ingest(*, offer_path: Path) -> Iterable[dict[str, Any]]:
    enums = load_enums()
    normalizer = load_region_normalizer()
    con = open_conn()
    con.execute(f"CREATE VIEW offer AS SELECT * FROM read_json_auto('{offer_path}')")

    for (
        sku_id, instance_type, region, engine_raw, depl_raw,
        license_model, vcpu_raw, memory_raw, usd, unit,
    ) in con.execute(_SQL).fetchall():
        if license_model and license_model != "No license required":
            continue
        if engine_raw not in _ENGINE_MAP or depl_raw not in _DEPL_MAP:
            continue
        region_normalized = normalizer.normalize(_PROVIDER, region)
        terms = {
            "commitment": "on_demand",
            "tenancy": _ENGINE_MAP[engine_raw],
            "os": _DEPL_MAP[depl_raw],
            "support_tier": "",
            "upfront": "",
            "payment_option": "",
        }
        apply_kind_defaults(_KIND, terms, enums)
        yield {
            "sku_id": sku_id,
            "provider": _PROVIDER,
            "service": _SERVICE,
            "kind": _KIND,
            "resource_name": instance_type,
            "region": region,
            "region_normalized": region_normalized,
            "terms_hash": terms_hash(terms),
            "resource_attrs": {
                "vcpu": int(vcpu_raw) if vcpu_raw else None,
                "memory_gb": _parse_memory(memory_raw) if memory_raw else None,
                "extra": {
                    "engine": _ENGINE_MAP[engine_raw],
                    "deployment_option": _DEPL_MAP[depl_raw],
                },
            },
            "terms": terms,
            "prices": [
                {"dimension": "compute", "tier": "", "amount": usd, "unit": unit.lower()},
            ],
        }

// VMFilter captures the flags `sku aws ec2 price/list` exposes.
type VMFilter struct {
	Provider     string
	Service      string
	InstanceType string
	Region       string
	Terms        Terms // populated with defaults at the cmd layer before calling
}

// DBRelationalFilter captures the flags `sku aws rds price/list` exposes.
type DBRelationalFilter struct {
	Provider         string
	Service          string
	InstanceType     string
	Region           string
	Terms            Terms // tenancy=engine, os=deployment-option
}

// LookupVM runs the compute.vm point lookup / list query.
func (c *Catalog) LookupVM(ctx context.Context, f VMFilter) ([]Row, error) {
	if f.InstanceType == "" {
		return nil, fmt.Errorf("catalog: LookupVM requires InstanceType")
	}
	return c.lookupResource(ctx, "compute.vm", f.Provider, f.Service,
		f.InstanceType, f.Region, f.Terms)
}

// LookupDBRelational runs the db.relational point lookup / list query.
func (c *Catalog) LookupDBRelational(ctx context.Context, f DBRelationalFilter) ([]Row, error) {
	if f.InstanceType == "" {
		return nil, fmt.Errorf("catalog: LookupDBRelational requires InstanceType")
	}
	return c.lookupResource(ctx, "db.relational", f.Provider, f.Service,
		f.InstanceType, f.Region, f.Terms)
}

// lookupResource is the shared scan path for cloud kinds. Region is
// optional (empty region is ambiguous for cross-region list queries,
// so an empty Region returns every region's row for that instance type).
// terms_hash is computed client-side from f.Terms before the query so
// the planner uses the idx_skus_lookup (resource_name, region, terms_hash)
// index; see spec §5.
func (c *Catalog) lookupResource(
	ctx context.Context,
	kind, provider, service, resourceName, region string, terms Terms,
) ([]Row, error) {
	var where []string
	var args []any
	where = append(where, "s.kind = ?")
	args = append(args, kind)
	where = append(where, "s.resource_name = ?")
	args = append(args, resourceName)
	if provider != "" {

LEFT JOIN resource_attrs ra ON ra.sku_id = s.sku_id

func shardMissingErr(shard string) *skuerrors.E {
	return &skuerrors.E{
		Code:       skuerrors.CodeNotFound,
		Message:    fmt.Sprintf("%s shard not installed", shard),
		Suggestion: fmt.Sprintf("Run: sku update %s", shard),
		Details: map[string]any{
			"shard":        shard,
			"install_hint": "sku update " + shard,
		},
	}
}

func applyStaleGate(cmd *cobra.Command, cat *catalog.Catalog, shard string, s config.Settings) error {
	age := cat.Age(time.Now().UTC())
	if s.StaleErrorDays > 0 && age >= s.StaleErrorDays && !s.StaleOK {
		e := &skuerrors.E{
			Code:       skuerrors.CodeStaleData,
			Message:    fmt.Sprintf("catalog %d days old exceeds threshold %d", age, s.StaleErrorDays),
			Suggestion: "Run: sku update " + shard,
			Details: map[string]any{
				"shard":          shard,
				"age_days":       age,
				"threshold_days": s.StaleErrorDays,
			},
		}
		skuerrors.Write(cmd.ErrOrStderr(), e)
		return e
	}
	if s.StaleWarningDays > 0 && age >= s.StaleWarningDays && !s.StaleOK {
		_, _ = fmt.Fprintf(cmd.ErrOrStderr(),
			"warning: catalog is %d days old (warn threshold %d); run `sku update %s`\n",
			age, s.StaleWarningDays, shard)
	}
	return nil
}

func renderRows(cmd *cobra.Command, rows []catalog.Row, s config.Settings) error {
	opts := output.Options{
		Preset:            output.Preset(s.Preset),
		Format:            s.Format,
		Pretty:            s.Pretty,
		Fields:            s.Fields,
		JQ:                s.JQ,
		IncludeRaw:        s.IncludeRaw,
		IncludeAggregated: s.IncludeAggregated,
		NoColor:           s.NoColor,
	}
	w := cmd.OutOrStdout()
	for _, r := range rows {
		b, err := output.Pipeline(r, opts)
		if err == output.ErrDropped {
			continue
		}
		if err != nil {
			wrapped := fmt.Errorf("render: %w", err)
			skuerrors.Write(cmd.ErrOrStderr(), wrapped)
			return wrapped
		}
		if _, wErr := w.Write(b); wErr != nil {
			return wErr
		}
	}
	return nil
}
