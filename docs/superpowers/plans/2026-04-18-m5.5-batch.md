# M5.5 — `sku batch` (NDJSON/array dispatcher + handler registry) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Introduce a typed in-process handler registry (`internal/batch`) and a new `sku batch` leaf that reads stdin (NDJSON or JSON array, auto-detected), dispatches each op to a registered handler, and emits one result record per op with a correct aggregated exit code. Ship with handlers for `llm price`, `aws ec2 price`, `compare`, and `estimate` — enough to satisfy the spec §9 exit criterion *"batch handles 50-op test with correct aggregated exit code"* while proving the pattern. Remaining leaf handlers (aws rds, s3, lambda, ebs, dynamodb, cloudfront; azure vm/sql/blob/functions/disks; gcp gce/cloud-sql/gcs/run/functions; search; llm price aliases for list/update/schema) are explicitly deferred to a follow-up plan (`m5.6-batch-handler-rollout`) so this session stays session-sized.

**Non-goals (deferred to m5.6 and beyond):**
- Registering the remaining ~20 leaf command handlers; m5.5 locks the pattern and leaves the rollout mechanical.
- `--concurrency N` (post-v1 per spec §4).
- Per-shard dedup fetch at the head of the batch (spec §3 §4 mentions; requires updater wiring deferred with the rest of M5).
- `#`-prefixed comments are implemented (spec §4), but multi-line comments and JSONC are not.
- Any change to the standalone-command code paths beyond extracting a reusable `Handler` for each of the four MVP leaves. Existing Cobra flag surfaces, error envelopes, and JSON outputs stay byte-identical.

**Architecture:**

1. **`internal/batch/registry.go`** — new package. Declares:
   - `type Handler func(ctx context.Context, args map[string]any, env Env) (result any, err error)` — `result` is the *raw Go value* that the standalone command would marshal to stdout (e.g. `[]catalog.Row`, `*estimate.Result`); the batch runner marshals it once into the op record. `err` is the canonical error the standalone command returned — the runner boxes it into an `*skuerrors.E` envelope with the same mapping as `skuerrors.Write`.
   - `type Env struct { Settings *Settings; Stdout, Stderr io.Writer; Profile string }` — holds the resolved global settings (preset, format, stale thresholds, include-raw, include-aggregated, verbose, dry-run) that would otherwise come from `globalSettings(cmd)`. Handlers ignore `Stdout`/`Stderr` in v1 because the runner collects the returned `result` directly; the writers exist so future streaming can plug in.
   - `Settings` is re-exported from `cmd/sku` via a duck-typed anonymous struct to avoid an import cycle: `internal/batch` depends on nothing from `cmd/sku`. Concretely, `batch.Env.Settings` is `*batch.Settings`, a copy of the fields `cmd/sku` already passes through `globalSettings`. `cmd/sku` constructs it when invoking dispatch and handler functions read what they need.
   - `var registry = map[string]Handler{}` and `func Register(name string, h Handler)` (panics on duplicate; test helper `ResetForTest(t *testing.T)` snapshots + restores). `func Lookup(name string) (Handler, bool)`.
   - Canonical names match spec §4: `"aws ec2 price"`, `"aws ec2 list"`, `"compare"`, `"estimate"`, `"llm price"`, etc. — space-separated full command path, no flags.

2. **`internal/batch/op.go`** — new file. Declares:
   - `type Op struct { Command string; Args map[string]any; Preset, Profile, JQ, Fields string; IncludeRaw, IncludeAggregated, Pretty bool; Format string }` — the shape accepted on stdin. Per-op flags override global settings for that op only.
   - `type Record struct { Index int; ExitCode int; Output any; Error *skuerrors.E }` — one per op. The runner marshals `Record` directly to JSON; `Output` is `any` so handlers can return typed results without the runner knowing the type.
   - `func ApplyOverrides(base batch.Settings, op Op) batch.Settings` — returns a per-op copy of the resolved settings with the op-level overrides layered on top (preset/profile/jq/fields/format/pretty/include_raw/include_aggregated).

3. **`internal/batch/parse.go`** — new file. Stdin parser:
   - `func Parse(r io.Reader) (ops []Op, format Format, err error)` where `Format` is `FormatArray` or `FormatNDJSON`. Auto-detects by peeking first non-whitespace byte via `bufio.Reader`.
   - `[` → `json.Decoder.Decode` into `[]Op`. Disallow unknown fields via `dec.DisallowUnknownFields()` so typos in per-op keys (`"commnad"`) surface as exit 4.
   - `{` → line-by-line via `bufio.Scanner`; skip blank lines and lines whose first non-whitespace byte is `#`. Each accepted line must decode to exactly one `Op` (also with unknown-field rejection); trailing junk on a line is an error.
   - Any other first byte: returns a sentinel `ErrBadFormat` that the Cobra layer maps to `skuerrors.Validation("flag_invalid", "stdin", "", ...)` → exit 4.
   - `Scanner` buffer is raised to 1 MiB (`MaxScanTokenSize` default) so large inline ops parse; anything bigger fails with a sentinel `ErrLineTooLong` that maps to exit 4.

4. **`internal/batch/dispatch.go`** — new file. Sequential dispatcher:
   - `func Dispatch(ctx context.Context, ops []Op, env Env) []Record` — iterates ops; for each op looks up the handler, calls it with a per-op `Env` produced by `ApplyOverrides`, converts any error via `boxErr` (a local helper that unwraps `*skuerrors.E` via `errors.As` and boxes other errors as `CodeGeneric`), and appends a `Record`.
   - Handler-not-found: `Record.Error = skuerrors.Validation("unknown_command", "command", op.Command, "Run `sku schema --list-commands` to see supported batch commands")`, exit 4.
   - Context cancellation short-circuits remaining ops with `CodeServer` envelopes so `ctx.Err()` propagates cleanly.
   - `func AggregateExit(records []Record) int` — returns `max(exit_code)` per spec §4, with `CodeOK` < all others.

5. **`internal/batch/fuzz_test.go`** — `FuzzBatchParse` per spec §6.5; seeds a few hand-picked lines and arrays, asserts `Parse` never panics and either returns an error or returns a `len(ops) >= 0` result.

6. **`cmd/sku/handlers.go`** — new file. Contains the MVP handler registrations that `Execute()` depends on at binary start:

   ```go
   package sku

   import (
       "github.com/sofq/sku/internal/batch"
   )

   func init() {
       batch.Register("llm price",     handleLLMPrice)
       batch.Register("aws ec2 price", handleAWSEC2Price)
       batch.Register("aws ec2 list",  handleAWSEC2List)
       batch.Register("compare",       handleCompare)
       batch.Register("estimate",      handleEstimate)
   }
   ```

   Each handler is a thin function extracted from the existing Cobra `RunE` body. The refactor preserves the existing `RunE` bodies unchanged — `RunE` now calls the handler and then runs the output pipeline exactly as before. Concretely: `runAWSCompute` (aws ec2 price/list) splits into `handleAWSEC2Price(ctx, args, env) ([]catalog.Row, error)` + an `emitComputeRows` tail shared by both call sites; the Cobra side still produces byte-identical output because it keeps driving `output.Pipeline` itself. Handlers return the raw `[]catalog.Row` (or `*estimate.Result`) so the batch runner marshals to the op record directly — this matches spec §4's "`output` is the op's JSON result" contract.

   `handleLLMPrice` mirrors the existing `newLLMPriceCmd` body up through `cat.LookupLLM`: shard-missing → `CodeNotFound`, stale-error-days check, return rows. `handleCompare` and `handleEstimate` follow the same shape against their existing bodies (`cmd/sku/compare.go`, `cmd/sku/estimate.go`). `args` keys are the spec §4 argument names (`model`, `serving_provider`, `instance_type`, `region`, `kind`, `vcpu`, `memory`, `items`, ...). A new `cmd/sku/handlers_args.go` keeps the `map[string]any` → typed-arg decoding in one place, using `mapstructure` *only* if it's already in `go.mod`; otherwise a small hand-rolled `argString(args, "model")` / `argFloat(args, "vcpu")` / `argStringSlice(args, "items")` suffices (grep `go.mod` first; do not add a new dep for this plan).

7. **`cmd/sku/batch.go`** — new Cobra leaf:
   - `sku batch [--format array|ndjson] [--input <path>]` (flags in addition to the persistent globals). `--input` defaults to stdin and exists for tests; the spec only documents stdin.
   - Reads the stream, calls `batch.Parse`, then `batch.Dispatch`, then marshals each `Record` back out in the *same* shape as the input format: array mode emits one JSON array; NDJSON mode emits one record per line. Pretty-print honors the global `--pretty`, applied line-by-line in NDJSON (each line individually indented) and once at the top level in array mode.
   - Process exit code = `batch.AggregateExit(records)`. Per-op stderr hints (stale-warning text that standalone commands write to stderr) are folded into `Record.Error.Details["warnings"]` for that op only (spec §4); `cmd/sku/batch.go` never writes to its own stderr except for batch-level parse/format failures.
   - Registered on the root command tree alongside the existing `estimate`, `compare`, `search` leaves (see `cmd/sku/root.go`).

8. **`cmd/sku/batch_test.go`** — end-to-end tests:
   - Array happy-path with three ops spanning llm/compute/compare; asserts JSON shape + exit 0.
   - NDJSON happy-path with 50 ops (25 of them deliberately bad `aws ec2 price` lookups that return `CodeNotFound`); asserts aggregate exit 3 per §4 (`max(0, 3) = 3`), that the process stderr is empty, and that every record carries a `CodeNotFound` envelope.
   - Bad first byte (`"abc"`); asserts exit 4, stderr carries a validation envelope, stdout is empty.
   - Unknown command name; asserts that op's `error.code == "validation"` and `error.details.reason == "unknown_command"`.
   - Mixed per-op `preset` override: verifies that overriding `preset` on one op does not affect adjacent ops (shared state bug sentinel).
   - `--pretty` in NDJSON mode produces multi-line records separated by a single `\n`.
   - Context cancellation: a cancelled parent context yields `CodeServer` envelopes for *un-started* ops while started ops finish normally (uses a test hook wrapping a registered handler).

9. **`internal/schema/commands.go`** + **`cmd/sku/schema.go`** — `sku schema --list-commands` already plans to enumerate batch-registered commands per spec §4; add a small shim that returns `batch.RegisteredNames()` (a new helper that returns the keys sorted lexicographically). This replaces any hard-coded list and keeps the schema in sync with the registry automatically.

```go
// Package batch provides the in-process handler registry and NDJSON/array
// dispatcher that powers `sku batch`. Handlers are registered by their
// canonical command name (space-separated, no flags) and are called directly
// from the batch runner with byte-identical error envelopes to the standalone
// path.
package batch

import (
	"context"
	"io"
	"sort"
	"sync"
	"testing"
)

// Settings mirrors the subset of cmd/sku global settings that handlers need.
// Kept in internal/batch to avoid an import cycle with cmd/sku.
type Settings struct {
	Preset            string
	Profile           string
	Format            string
	Pretty            bool
	JQ                string
	Fields            string
	IncludeRaw        bool
	IncludeAggregated bool
	StaleOK           bool
	StaleWarningDays  int
	StaleErrorDays    int
	DryRun            bool
	Verbose           bool
	NoColor           bool
}

// Env is the ambient context passed to a handler.
type Env struct {
	Settings *Settings
	Stdout   io.Writer
	Stderr   io.Writer
}

// Handler is the typed signature every batch-registered command implements.
// result is the raw Go value the standalone command would marshal to stdout.
type Handler func(ctx context.Context, args map[string]any, env Env) (result any, err error)

var (
	mu       sync.RWMutex
	registry = map[string]Handler{}
)

// Register associates name with h. Panics on duplicate.
func Register(name string, h Handler) {
	mu.Lock()
	defer mu.Unlock()
	if _, dup := registry[name]; dup {
		panic("batch: duplicate Register for " + name)
	}
	registry[name] = h
}

// Lookup returns the handler for name.
func Lookup(name string) (Handler, bool) {
	mu.RLock()
	defer mu.RUnlock()
	h, ok := registry[name]
	return h, ok
}

// RegisteredNames returns all registered command names in lex order.
func RegisteredNames() []string {
	mu.RLock()
	defer mu.RUnlock()
	out := make([]string, 0, len(registry))
	for k := range registry {
		out = append(out, k)
	}
	sort.Strings(out)
	return out
}

// ResetForTest snapshots and restores the registry. Test-only.
func ResetForTest(t *testing.T) {
	t.Helper()
	mu.Lock()
	snap := make(map[string]Handler, len(registry))
	for k, v := range registry {
		snap[k] = v
	}
	registry = map[string]Handler{}
	mu.Unlock()
	t.Cleanup(func() {
		mu.Lock()
		registry = snap
		mu.Unlock()
	})
}
```

```go
package batch

import skuerrors "github.com/sofq/sku/internal/errors"

// Op is one entry in the batch input stream. Per-op flags override the
// corresponding global setting for that op only.
type Op struct {
	Command           string         `json:"command"`
	Args              map[string]any `json:"args,omitempty"`
	Preset            string         `json:"preset,omitempty"`
	Profile           string         `json:"profile,omitempty"`
	JQ                string         `json:"jq,omitempty"`
	Fields            string         `json:"fields,omitempty"`
	Format            string         `json:"format,omitempty"`
	Pretty            bool           `json:"pretty,omitempty"`
	IncludeRaw        bool           `json:"include_raw,omitempty"`
	IncludeAggregated bool           `json:"include_aggregated,omitempty"`
}

// Record is the per-op result emitted on stdout.
type Record struct {
	Index    int           `json:"index"`
	ExitCode int           `json:"exit_code"`
	Output   any           `json:"output"`
	Error    *skuerrors.E  `json:"error,omitempty"`
}

// ApplyOverrides returns a copy of base with op-level overrides applied.
func ApplyOverrides(base Settings, op Op) Settings {
	out := base
	if op.Preset != "" {
		out.Preset = op.Preset
	}
	if op.Profile != "" {
		out.Profile = op.Profile
	}
	if op.JQ != "" {
		out.JQ = op.JQ
	}
	if op.Fields != "" {
		out.Fields = op.Fields
	}
	if op.Format != "" {
		out.Format = op.Format
	}
	if op.Pretty {
		out.Pretty = true
	}
	if op.IncludeRaw {
		out.IncludeRaw = true
	}
	if op.IncludeAggregated {
		out.IncludeAggregated = true
	}
	return out
}
```

```go
package batch

import (
	"bufio"
	"bytes"
	"encoding/json"
	"errors"
	"fmt"
	"io"
	"strings"
)

// Format is the detected stdin format.
type Format int

const (
	FormatNDJSON Format = iota
	FormatArray
)

// ErrBadFormat is returned when the first non-whitespace byte is neither
// '[' nor '{'.
var ErrBadFormat = errors.New("batch: stdin must start with '[' (array) or '{' (NDJSON)")

// ErrLineTooLong is returned when a single NDJSON line exceeds 1 MiB.
var ErrLineTooLong = errors.New("batch: NDJSON line exceeds 1 MiB limit")

const ndjsonMaxLine = 1 << 20

// Parse reads stdin and returns the parsed ops plus the detected format.
func Parse(r io.Reader) ([]Op, Format, error) {
	br := bufio.NewReader(r)
	// Skip leading whitespace to find the first meaningful byte.
	var first byte
	for {
		b, err := br.ReadByte()
		if err == io.EOF {
			return nil, FormatNDJSON, nil // empty input: zero ops, NDJSON default
		}
		if err != nil {
			return nil, 0, err
		}
		if b == ' ' || b == '\t' || b == '\n' || b == '\r' {
			continue
		}
		first = b
		_ = br.UnreadByte()
		break
	}

	switch first {
	case '[':
		dec := json.NewDecoder(br)
		dec.DisallowUnknownFields()
		var ops []Op
		if err := dec.Decode(&ops); err != nil {
			return nil, 0, fmt.Errorf("array decode: %w", err)
		}
		return ops, FormatArray, nil
	case '{':
		scanner := bufio.NewScanner(br)
		scanner.Buffer(make([]byte, 0, 64*1024), ndjsonMaxLine)
		var ops []Op
		for scanner.Scan() {
			line := strings.TrimSpace(scanner.Text())
			if line == "" || strings.HasPrefix(line, "#") {
				continue
			}
			dec := json.NewDecoder(bytes.NewReader([]byte(line)))
			dec.DisallowUnknownFields()
			var op Op
			if err := dec.Decode(&op); err != nil {
				return nil, 0, fmt.Errorf("ndjson decode line %d: %w", len(ops)+1, err)
			}
			if dec.More() {
				return nil, 0, fmt.Errorf("ndjson line %d has trailing junk", len(ops)+1)
			}
			ops = append(ops, op)
		}
		if err := scanner.Err(); err != nil {
			if errors.Is(err, bufio.ErrTooLong) {
				return nil, 0, ErrLineTooLong
			}
			return nil, 0, err
		}
		return ops, FormatNDJSON, nil
	default:
		return nil, 0, ErrBadFormat
	}
}
```

```go
package batch

import (
	"context"
	"errors"

	skuerrors "github.com/sofq/sku/internal/errors"
)

// Dispatch runs ops sequentially against the registry. Errors become per-op
// Record.Error envelopes; the process stderr stays clean.
func Dispatch(ctx context.Context, ops []Op, env Env) []Record {
	out := make([]Record, 0, len(ops))
	for i, op := range ops {
		if err := ctx.Err(); err != nil {
			out = append(out, Record{
				Index: i, ExitCode: skuerrors.CodeServer.ExitCode(),
				Error: &skuerrors.E{Code: skuerrors.CodeServer, Message: err.Error()},
			})
			continue
		}
		h, ok := Lookup(op.Command)
		if !ok {
			e := skuerrors.Validation("unknown_command", "command", op.Command,
				"Run `sku schema --list-commands` to see supported batch commands")
			out = append(out, Record{Index: i, ExitCode: e.Code.ExitCode(), Error: e})
			continue
		}
		perOp := ApplyOverrides(*env.Settings, op)
		result, err := h(ctx, op.Args, Env{Settings: &perOp, Stdout: env.Stdout, Stderr: env.Stderr})
		if err != nil {
			e := boxErr(err)
			out = append(out, Record{Index: i, ExitCode: e.Code.ExitCode(), Error: e})
			continue
		}
		out = append(out, Record{Index: i, ExitCode: 0, Output: result})
	}
	return out
}

// AggregateExit returns max(exit_code) across records; 0 on empty.
func AggregateExit(recs []Record) int {
	worst := 0
	for _, r := range recs {
		if r.ExitCode > worst {
			worst = r.ExitCode
		}
	}
	return worst
}

func boxErr(err error) *skuerrors.E {
	var e *skuerrors.E
	if errors.As(err, &e) {
		return e
	}
	return &skuerrors.E{Code: skuerrors.CodeGeneric, Message: err.Error()}
}
```

```go
package sku

import "strconv"

func argString(args map[string]any, key string) string {
	if v, ok := args[key].(string); ok {
		return v
	}
	return ""
}

func argFloat(args map[string]any, key string) (float64, bool) {
	switch v := args[key].(type) {
	case float64:
		return v, true
	case int:
		return float64(v), true
	case string:
		f, err := strconv.ParseFloat(v, 64)
		if err != nil {
			return 0, false
		}
		return f, true
	}
	return 0, false
}

func argStringSlice(args map[string]any, key string) []string {
	raw, ok := args[key].([]any)
	if !ok {
		return nil
	}
	out := make([]string, 0, len(raw))
	for _, v := range raw {
		if s, ok := v.(string); ok {
			out = append(out, s)
		}
	}
	return out
}

func argBool(args map[string]any, key string) bool {
	b, _ := args[key].(bool)
	return b
}
```

```go
package sku

import (
	"context"
	"encoding/json"
	"fmt"

	"github.com/spf13/cobra"

	"github.com/sofq/sku/internal/batch"
	skuerrors "github.com/sofq/sku/internal/errors"
)

func newBatchCmd() *cobra.Command {
	c := &cobra.Command{
		Use:   "batch",
		Short: "Run multiple sku ops from stdin (NDJSON or JSON array)",
		RunE: func(cmd *cobra.Command, _ []string) error {
			g := globalSettings(cmd)
			settings := ToBatchSettings(g)

			ops, format, err := batch.Parse(cmd.InOrStdin())
			if err != nil {
				e := skuerrors.Validation("flag_invalid", "stdin", "", err.Error())
				skuerrors.Write(cmd.ErrOrStderr(), e)
				return e
			}

			env := batch.Env{Settings: &settings, Stdout: cmd.OutOrStdout(), Stderr: cmd.ErrOrStderr()}
			records := batch.Dispatch(context.Background(), ops, env)

			if err := emitBatch(cmd, format, records, g.Pretty); err != nil {
				return fmt.Errorf("batch: emit: %w", err)
			}

			code := batch.AggregateExit(records)
			if code == 0 {
				return nil
			}
			// Surface aggregate exit without re-writing stderr; per-op errors already live in the records.
			return &skuerrors.E{Code: exitCodeToErrCode(code), Message: "batch completed with per-op errors"}
		},
	}
	return c
}

func emitBatch(cmd *cobra.Command, f batch.Format, recs []batch.Record, pretty bool) error {
	w := cmd.OutOrStdout()
	switch f {
	case batch.FormatArray:
		enc := json.NewEncoder(w)
		if pretty {
			enc.SetIndent("", "  ")
		}
		return enc.Encode(recs)
	case batch.FormatNDJSON:
		for _, r := range recs {
			enc := json.NewEncoder(w)
			if pretty {
				enc.SetIndent("", "  ")
			}
			if err := enc.Encode(r); err != nil {
				return err
			}
		}
		return nil
	default:
		return fmt.Errorf("unknown format %v", f)
	}
}

// exitCodeToErrCode maps an exit code int back to the canonical Code so the
// batch wrapper can return one structured error that carries the aggregated
// severity to Execute().
func exitCodeToErrCode(exit int) skuerrors.Code {
	switch exit {
	case 2:
		return skuerrors.CodeAuth
	case 3:
		return skuerrors.CodeNotFound
	case 4:
		return skuerrors.CodeValidation
	case 5:
		return skuerrors.CodeRateLimited
	case 6:
		return skuerrors.CodeConflict
	case 7:
		return skuerrors.CodeServer
	case 8:
		return skuerrors.CodeStaleData
	}
	return skuerrors.CodeGeneric
}
```
