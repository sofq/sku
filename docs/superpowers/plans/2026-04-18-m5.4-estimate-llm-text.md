# M5.4 — `sku estimate` llm.text estimator (openrouter shard) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add an `llm.text` estimator to the `internal/estimate` registry so workloads like `sku estimate --item llm:anthropic/claude-opus-4.6:input=1M:output=500K` return a `monthly_usd` computed from the openrouter shard's `prompt` + `completion` per-token prices. After this plan, the inline DSL (m5.1), `--config` YAML/JSON (m5.2), and `--stdin` workloads all accept `llm.text` items alongside existing `compute.vm` and `storage.object` items. Non-goals (deferred): `sku batch` (tracked for a later m5.x plan); `db.relational`, `compute.function`, `storage.block`, `network.cdn` estimators; cached-input / cached-output pricing dimensions; per-request overhead math; `--currency` native conversion.

**Architecture:**

1. **`internal/estimate/item.go`** — extend `ParseItem` with a special branch: if the first colon-separated segment is exactly `"llm"` (no slash required), treat the raw form as `llm:<model>:[k=v]...`. `<model>` is the second segment and may itself contain a `/` (e.g. `anthropic/claude-opus-4.6`). The returned `Item` carries `Provider="llm"`, `Service="text"`, `Resource=<model>`, `Kind="llm.text"`. Params parse identically to the existing form. The existing `provider/service:resource...` branch stays the default for everything else, so no behavior changes for `aws/ec2`, `aws/s3`, etc.

2. **`internal/estimate/kinds.go`** — add one entry `"llm/text": "llm.text"` so the workload-doc path (`DecodeWorkload` in `workload.go`) resolves kind the same way it does for compute/storage items when callers pass `provider: "llm"`, `service: "text"`. The inline DSL path consults the kind-map only via `ParseItem`; the llm branch above sets `Kind` directly, so this map entry exists purely for the YAML/JSON/stdin code path.

3. **`internal/estimate/llm.go`** — new file. Declares:
   - `var lookupLLM = func(ctx context.Context, shard string, f catalog.LLMFilter) ([]catalog.Row, error) { ... }` — the test seam, symmetric with `lookupVM` / `lookupStorageObject`. Default body calls `catalog.Open(catalog.ShardPath(shard))` then `cat.LookupLLM`.
   - Constant `llmShard = "openrouter"` — all llm.text items live in the single openrouter shard in v1.
   - `llmPromptDims = map[string]bool{"prompt": true, "input": true, "input_tokens": true}` and `llmCompletionDims = map[string]bool{"completion": true, "output": true, "output_tokens": true}` — dimension aliases. The seed + M1 shard store `prompt` / `completion`; the aliases absorb naming drift if the pipeline ever renames.
   - `type llmTextEstimator struct{}` with `Kind() == "llm.text"` and `Estimate(ctx, Item) (LineItem, error)`.
     - Required params: `input` (monthly prompt tokens) and `output` (monthly completion tokens), both parsed via `parseQuantity` (so `1M` / `500K` / `1000` all work). A missing param is treated as `0`, but if both are `0` the estimator returns `"estimate/llm.text: %q requires :input=<tokens> and/or :output=<tokens>"`.
     - Optional params: `serving_provider` (disambiguates multi-provider models; passed into `catalog.LLMFilter.ServingProvider`) and `requests` (parsed only to store in `LineItem.Notes`; not used in cost math).
     - `IncludeAggregated` defaults to `false` so the aggregated `openrouter` row does not double-count with first-party rows. If the caller explicitly sets `serving_provider=openrouter`, set `IncludeAggregated=true` so that row surfaces.
     - On zero rows: `"estimate/llm.text: no SKU for model %q (serving_provider=%q)"`; on >1 rows without `serving_provider`: `"estimate/llm.text: %d rows for %q — add :serving_provider=<name>"`.
     - Price extraction: scan `row.Prices`, match dimensions against the alias sets. A missing `prompt` price is an error only when `input > 0`; a missing `completion` price is an error only when `output > 0`. Tier is ignored in v1 (all v1 prices use `tier=''`).
     - `monthly := input*promptPrice + output*completionPrice` (prices are USD/token, so quantities are token counts).
     - `LineItem` shape: `HourlyUSD = 0`, `Quantity = input + output`, `QuantityUnit = "token"`, `Notes` holds the serving provider actually resolved (e.g. `"serving_provider=anthropic"`) plus `"requests=%g"` when the param was supplied.
   - `func init() { Register(llmTextEstimator{}) }`.

4. **`internal/estimate/item_test.go`** — new cases cover the llm branch: happy parse of `llm:anthropic/claude-opus-4.6:input=1M:output=500K`, model with slash is preserved in `Resource`, missing model (`llm::input=1M`) errors, params after the model parse into the map unchanged.

```go
func ParseItem(raw string) (Item, error) {
	if raw == "" {
		return Item{}, fmt.Errorf("estimate/item: empty value")
	}
	segs := strings.Split(raw, ":")
	if len(segs) < 2 {
		return Item{}, fmt.Errorf("estimate/item: %q missing ':resource'", raw)
	}
	ps := segs[0]

	// Special form: "llm:<model>[:k=v...]". Model may contain "/".
	if strings.ToLower(ps) == "llm" {
		model := segs[1]
		if model == "" {
			return Item{}, fmt.Errorf("estimate/item: %q: empty model after llm:", raw)
		}
		params, err := parseItemParams(raw, segs[2:])
		if err != nil {
			return Item{}, err
		}
		return Item{
			Raw: raw, Provider: "llm", Service: "text",
			Resource: model, Kind: "llm.text", Params: params,
		}, nil
	}

	// Default form: "provider/service:resource[:k=v...]".
	slash := strings.IndexByte(ps, '/')
	if slash <= 0 || slash == len(ps)-1 {
		return Item{}, fmt.Errorf("estimate/item: %q: first segment must be provider/service", raw)
	}
	provider := strings.ToLower(ps[:slash])
	service := strings.ToLower(ps[slash+1:])
	resource := segs[1]
	if resource == "" {
		return Item{}, fmt.Errorf("estimate/item: %q: empty resource", raw)
	}
	kind, ok := providerServiceKind[provider+"/"+service]
	if !ok {
		return Item{}, fmt.Errorf("estimate/item: unsupported provider/service %q", provider+"/"+service)
	}
	params, err := parseItemParams(raw, segs[2:])
	if err != nil {
		return Item{}, err
	}
	return Item{
		Raw: raw, Provider: provider, Service: service,
		Resource: resource, Kind: kind, Params: params,
	}, nil
}

func parseItemParams(raw string, kvs []string) (map[string]string, error) {
	params := map[string]string{}
	for _, kv := range kvs {
		eq := strings.IndexByte(kv, '=')
		if eq <= 0 {
			return nil, fmt.Errorf("estimate/item: %q: param %q must be key=value", raw, kv)
		}
		k := strings.ToLower(kv[:eq])
		v := kv[eq+1:]
		if _, exists := params[k]; exists {
			return nil, fmt.Errorf("estimate/item: %q: duplicate param %q", raw, k)
		}
		params[k] = v
	}
	return params, nil
}
```

```go
package estimate

import (
	"context"
	"fmt"
	"strings"

	"github.com/sofq/sku/internal/catalog"
)

const llmShard = "openrouter"

var lookupLLM = func(ctx context.Context, shard string, f catalog.LLMFilter) ([]catalog.Row, error) {
	cat, err := catalog.Open(catalog.ShardPath(shard))
	if err != nil {
		return nil, err
	}
	defer func() { _ = cat.Close() }()
	return cat.LookupLLM(ctx, f)
}

var (
	llmPromptDims     = map[string]bool{"prompt": true, "input": true, "input_tokens": true}
	llmCompletionDims = map[string]bool{"completion": true, "output": true, "output_tokens": true}
)

type llmTextEstimator struct{}

func (llmTextEstimator) Kind() string { return "llm.text" }

func (llmTextEstimator) Estimate(ctx context.Context, it Item) (LineItem, error) {
	inTokens, err := paramFloatLLM(it.Params, "input", 0, 0)
	if err != nil {
		return LineItem{}, err
	}
	outTokens, err := paramFloatLLM(it.Params, "output", 0, 0)
	if err != nil {
		return LineItem{}, err
	}
	if inTokens == 0 && outTokens == 0 {
		return LineItem{}, fmt.Errorf("estimate/llm.text: %q requires :input=<tokens> and/or :output=<tokens>", it.Raw)
	}
	requests, err := paramFloatLLM(it.Params, "requests", 0, 0)
	if err != nil {
		return LineItem{}, err
	}
	serving := strings.ToLower(it.Params["serving_provider"])

	rows, err := lookupLLM(ctx, llmShard, catalog.LLMFilter{
		Model:             it.Resource,
		ServingProvider:   serving,
		IncludeAggregated: serving == "openrouter",
	})
	if err != nil {
		return LineItem{}, fmt.Errorf("estimate/llm.text: lookup: %w", err)
	}
	if len(rows) == 0 {
		return LineItem{}, fmt.Errorf("estimate/llm.text: no SKU for model %q (serving_provider=%q)", it.Resource, serving)
	}
	if len(rows) > 1 {
		return LineItem{}, fmt.Errorf("estimate/llm.text: %d rows for %q — add :serving_provider=<name>", len(rows), it.Resource)
	}
	r := rows[0]

	var promptPrice, completionPrice float64
	var havePrompt, haveCompletion bool
	for _, p := range r.Prices {
		switch {
		case llmPromptDims[p.Dimension]:
			promptPrice, havePrompt = p.Amount, true
		case llmCompletionDims[p.Dimension]:
			completionPrice, haveCompletion = p.Amount, true
		}
	}
	if inTokens > 0 && !havePrompt {
		return LineItem{}, fmt.Errorf("estimate/llm.text: no prompt price on SKU %s", r.SKUID)
	}
	if outTokens > 0 && !haveCompletion {
		return LineItem{}, fmt.Errorf("estimate/llm.text: no completion price on SKU %s", r.SKUID)
	}

	monthly := inTokens*promptPrice + outTokens*completionPrice

	notes := []string{fmt.Sprintf("serving_provider=%s", r.Provider)}
	if requests > 0 {
		notes = append(notes, fmt.Sprintf("requests=%g", requests))
	}

	return LineItem{
		Item: it, Kind: "llm.text", SKUID: r.SKUID,
		Provider: r.Provider, Service: r.Service, Resource: r.ResourceName,
		Region:       r.Region,
		Quantity:     inTokens + outTokens,
		QuantityUnit: "token",
		MonthlyUSD:   monthly,
		Notes:        notes,
	}, nil
}

func paramFloatLLM(p map[string]string, k string, def, minVal float64) (float64, error) {
	s, ok := p[k]
	if !ok {
		return def, nil
	}
	f, err := parseQuantity(s)
	if err != nil {
		return 0, fmt.Errorf("estimate/llm.text: %s=%q: %w", k, s, err)
	}
	if f < minVal {
		return 0, fmt.Errorf("estimate/llm.text: %s=%v below minimum %v", k, f, minVal)
	}
	return f, nil
}

func init() {
	Register(llmTextEstimator{})
}
```
