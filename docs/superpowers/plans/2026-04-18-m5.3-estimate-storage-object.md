# M5.3 — `sku estimate` storage.object estimator (aws/s3, azure/blob, gcp/gcs) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a `storage.object` estimator to the `internal/estimate` registry and wire `aws/s3`, `azure/blob`, `gcp/gcs` into the DSL kind map. After this plan, `sku estimate --item aws/s3:standard:region=us-east-1:gb_month=500:put_requests=10000:get_requests=100000` returns a `MonthlyUSD` driven by the `storage`, `requests-put`/`write-ops`, and `requests-get`/`read-ops` dimensions on the matching catalog row. The inline DSL, `--config`, and `--stdin` input forms from m5.1/m5.2 all work with the new kind for free because they dispatch through the shared `estimate.Run`. Non-goals (deferred): `db.relational`, `llm.text`, `compute.function`, `storage.block`, `network.cdn`, `db.nosql` estimators; `sku batch`; per-region fan-out; tiered-pricing math.

**Architecture:**

1. **`internal/estimate/kinds.go`** — extend `providerServiceKind` with three new entries mapping `"aws/s3"`, `"azure/blob"`, `"gcp/gcs"` to `"storage.object"`. `ParseItem` already consults this map; no parser change needed.

2. **`internal/estimate/storage_object.go`** — new file. Declares:
   - `var lookupStorageObject = func(ctx context.Context, shard string, f catalog.StorageObjectFilter) ([]catalog.Row, error) { ... }` — the test seam, symmetric with `lookupVM` in `vm.go`. Default body opens the shard via `catalog.Open(catalog.ShardPath(shard))` and calls `cat.LookupStorageObject`.
   - `providerServiceShardStorage = map[string]string{"aws/s3":"aws-s3","azure/blob":"azure-blob","gcp/gcs":"gcp-gcs"}`.
   - `storagePutDims = map[string]bool{"requests-put": true, "write-ops": true}` and `storageGetDims = map[string]bool{"requests-get": true, "read-ops": true}` — dimension aliases across providers. Storage dimension is always `"storage"` with unit `"gb-mo"`.
   - `type storageObjectEstimator struct{}` with `Kind() == "storage.object"` and `Estimate(ctx, Item) (LineItem, error)`. Required param: `region`. Optional params (all default to 0, must parse as non-negative floats via `parseQuantity`): `gb_month`, `put_requests`, `get_requests`. Resource (from the DSL) is the storage class. On lookup returning zero rows return `"estimate/storage.object: no SKU for %s/%s:%s in %s"`; on >1 rows return `"estimate/storage.object: ambiguous (%d rows) for ..."` — same shape as `vmEstimator`. Compute `monthly = gb_month*storage + put_requests*put_price + get_requests*get_price`, where each of the three dimension prices is resolved against the row's `Prices` slice (storage matched by `Dimension=="storage"`, put/get matched by the alias sets). A missing `storage` price row is an error only when `gb_month>0`; missing put/get prices are errors only when the corresponding param > 0. This keeps "storage-only" and "requests-only" estimates viable without forcing the shard to carry every dimension.
   - `func init() { Register(storageObjectEstimator{}) }`.
   - `LineItem` shape: `HourlyUSD = 0` (not applicable), `Quantity = gb_month`, `QuantityUnit = "gb-mo"` (so the JSON envelope surfaces storage volume as the primary quantity).

3. **`internal/estimate/storage_object_test.go`** — new file. Covers: happy path with storage-only, happy path with all three dimensions, alias coverage (`write-ops`/`read-ops`), zero-param short-circuit (no storage dimension needed if `gb_month=0`), required-region, not-found, ambiguous. Uses `lookupStorageObject` seam override and `resetRegistry(t)`.

4. **`internal/catalog/testdata/build.go`** — extend the public helper set with `BuildAWSS3Shard(dst string) error` that embeds `seed_aws_m3a2.sql` (which already seeds `s3-standard-*` rows with `storage`, `requests-put`, `requests-get` prices per §5). No new SQL file — the existing m3a.2 seed covers storage.object.

5. **`cmd/sku/testutil_estimate_test.go`** — add a sibling helper `testutilSeededEstimateCatalogS3(t)` that seeds both `aws-ec2.db` (already needed by VM tests co-located in the same file) and `aws-s3.db` under `SKU_DATA_DIR`.

6. **`cmd/sku/estimate_test.go`** — add `TestEstimate_storageObject_endToEnd` that drives `sku estimate --item aws/s3:standard:region=us-east-1:gb_month=500:put_requests=1000:get_requests=5000` through `root.Execute()`, asserts a non-zero `monthly_usd`, and checks the envelope carries `quantity_unit == "gb-mo"`.

7. **`cmd/sku/testdata/workload-storage.yaml`** — new fixture file exercising `--config` with a storage.object item. Add `TestEstimate_storageObject_configYAML` using the new fixture.

8. **`CLAUDE.md`** — append three storage.object examples to the "Quick path" block so agents discover the new surface.

```go
package estimate

import (
	"context"
	"fmt"

	"github.com/sofq/sku/internal/catalog"
)

var lookupStorageObject = func(ctx context.Context, shard string, f catalog.StorageObjectFilter) ([]catalog.Row, error) {
	cat, err := catalog.Open(catalog.ShardPath(shard))
	if err != nil {
		return nil, err
	}
	defer func() { _ = cat.Close() }()
	return cat.LookupStorageObject(ctx, f)
}

var providerServiceShardStorage = map[string]string{
	"aws/s3":     "aws-s3",
	"azure/blob": "azure-blob",
	"gcp/gcs":    "gcp-gcs",
}

var (
	storagePutDims = map[string]bool{"requests-put": true, "write-ops": true}
	storageGetDims = map[string]bool{"requests-get": true, "read-ops": true}
)

type storageObjectEstimator struct{}

func (storageObjectEstimator) Kind() string { return "storage.object" }

func (storageObjectEstimator) Estimate(ctx context.Context, it Item) (LineItem, error) {
	region := it.Params["region"]
	if region == "" {
		return LineItem{}, fmt.Errorf("estimate/storage.object: %q missing :region=<name>", it.Raw)
	}
	gbMonth, err := paramFloatStorage(it.Params, "gb_month", 0, 0)
	if err != nil {
		return LineItem{}, err
	}
	putReqs, err := paramFloatStorage(it.Params, "put_requests", 0, 0)
	if err != nil {
		return LineItem{}, err
	}
	getReqs, err := paramFloatStorage(it.Params, "get_requests", 0, 0)
	if err != nil {
		return LineItem{}, err
	}

	shard, ok := providerServiceShardStorage[it.Provider+"/"+it.Service]
	if !ok {
		return LineItem{}, fmt.Errorf("estimate/storage.object: no shard mapped for %s/%s", it.Provider, it.Service)
	}

	rows, err := lookupStorageObject(ctx, shard, catalog.StorageObjectFilter{
		Provider: it.Provider, Service: it.Service,
		StorageClass: it.Resource, Region: region,
		Terms: catalog.Terms{Commitment: "on_demand"},
	})
	if err != nil {
		return LineItem{}, fmt.Errorf("estimate/storage.object: lookup: %w", err)
	}
	if len(rows) == 0 {
		return LineItem{}, fmt.Errorf("estimate/storage.object: no SKU for %s/%s:%s in %s", it.Provider, it.Service, it.Resource, region)
	}
	if len(rows) > 1 {
		return LineItem{}, fmt.Errorf("estimate/storage.object: ambiguous (%d rows) for %s/%s:%s in %s", len(rows), it.Provider, it.Service, it.Resource, region)
	}
	r := rows[0]

	var storagePrice, putPrice, getPrice float64
	var haveStorage, havePut, haveGet bool
	for _, p := range r.Prices {
		switch {
		case p.Dimension == "storage":
			storagePrice, haveStorage = p.Amount, true
		case storagePutDims[p.Dimension]:
			putPrice, havePut = p.Amount, true
		case storageGetDims[p.Dimension]:
			getPrice, haveGet = p.Amount, true
		}
	}
	if gbMonth > 0 && !haveStorage {
		return LineItem{}, fmt.Errorf("estimate/storage.object: no storage (gb-mo) price on SKU %s", r.SKUID)
	}
	if putReqs > 0 && !havePut {
		return LineItem{}, fmt.Errorf("estimate/storage.object: no put-requests price on SKU %s", r.SKUID)
	}
	if getReqs > 0 && !haveGet {
		return LineItem{}, fmt.Errorf("estimate/storage.object: no get-requests price on SKU %s", r.SKUID)
	}

	monthly := gbMonth*storagePrice + putReqs*putPrice + getReqs*getPrice
	return LineItem{
		Item: it, Kind: "storage.object", SKUID: r.SKUID,
		Provider: r.Provider, Service: r.Service, Resource: r.ResourceName,
		Region:   r.Region,
		Quantity: gbMonth, QuantityUnit: "gb-mo",
		MonthlyUSD: monthly,
	}, nil
}

func paramFloatStorage(p map[string]string, k string, def, minVal float64) (float64, error) {
	s, ok := p[k]
	if !ok {
		return def, nil
	}
	f, err := parseQuantity(s)
	if err != nil {
		return 0, fmt.Errorf("estimate/storage.object: %s=%q: %w", k, s, err)
	}
	if f < minVal {
		return 0, fmt.Errorf("estimate/storage.object: %s=%v below minimum %v", k, f, minVal)
	}
	return f, nil
}

func init() {
	Register(storageObjectEstimator{})
}
```
