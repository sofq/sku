# M3b.4 — GCP Cloud Storage + Cloud Run + Cloud Functions Shards (on-demand) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship the remaining three core GCP shards — `gcp-gcs`, `gcp-run`, `gcp-functions`, on-demand only — end-to-end: Python ingest from the Google Cloud Billing Catalog API JSON (fixtures in tests), SQLite packaging via the shared packager, the existing pure-Go `LookupStorageObject` / `LookupServerlessFunction` paths reused, and six new Cobra leaves (`sku gcp gcs {price,list}`, `sku gcp run {price,list}`, `sku gcp functions {price,list}`) returning valid §4 envelopes. `internal/updater.DefaultSources` gains entries for the three new shards. The daily CI pipeline + validation-harness extension to GCP stays deferred past m3b.4.

**Architecture:**

1. **Pipeline half (`pipeline/ingest/gcp_gcs.py`, `pipeline/ingest/gcp_run.py`, `pipeline/ingest/gcp_functions.py`):** the shape established in m3b.3 (`gcp_gce.py` + `gcp_cloud_sql.py`) is reused unchanged. Each new module reads a snapshot of the Cloud Billing Catalog API JSON payload (`{"skus": [...]}` array where each entry has `{name, skuId, description, category:{serviceDisplayName,resourceFamily,resourceGroup,usageType}, serviceRegions:[...], pricingInfo:[{pricingExpression:{usageUnit, tieredRates:[{unitPrice:{units,nanos,currencyCode}}]}}]}`). DuckDB's `read_json_auto` lifts the array; Python applies the per-shard filters (`usageType='OnDemand'`, currency USD, region-in-normalizer, `serviceDisplayName` allowlist). Prices are micros (`units + nanos/1e9`). `terms_hash` is computed via the existing `terms_hash()` helper in `pipeline/normalize/terms.py` — no UDF change. Shared helpers in `pipeline/ingest/gcp_common.py` (added in m3b.3) cover region normalization + price decoding + usage-unit canonicalization.
2. **Packager (`pipeline/package/build_shard.py` — unchanged):** NDJSON rows → SQLite shard with the full §5 schema + metadata. Identical pathway used by every M3a.* + M3b.* shard.
3. **Client half (pure Go):** `internal/catalog.LookupStorageObject` (kind=`storage.object`) covers `gcp-gcs`; `internal/catalog.LookupServerlessFunction` (kind=`compute.function`) covers both `gcp-run` and `gcp-functions`. Both helpers already accept arbitrary provider/service strings and the shared terms struct — no Go signature change. `cmd/sku/gcp.go` registers three new leaves under the existing `sku gcp` parent; `cmd/sku/gcp_gcs.go` mirrors `cmd/sku/azure_blob.go`, `cmd/sku/gcp_run.go` and `cmd/sku/gcp_functions.go` mirror `cmd/sku/azure_functions.go`, swapping the shard constant + the default flag values. `internal/updater.DefaultSources` gains three entries pointing at `data-bootstrap-gcp-gcs` / `data-bootstrap-gcp-run` / `data-bootstrap-gcp-functions` release tags.

**Tech Stack:** Python 3.11+, existing `duckdb>=0.10` (no new dep), existing `pipeline/package/build_shard.py` (unchanged), Go 1.25, `modernc.org/sqlite`, `github.com/spf13/cobra`. No new Go deps.

**Spec ref:** `docs/superpowers/specs/2026-04-18-sku-design.md` §3 (format stack, shard granularity, update flow), §4 (command tree, global flags, output schema, lazy shard fetch), §5 (kind taxonomy `storage.object`+`compute.function`, terms_hash, region normalization, indexes), §6 (unit/integration test targets), §9 M3b entry.

**Kind-taxonomy note for Cloud Run (pragmatic deferral).** Cloud Run is listed as `compute.container` in §5's taxonomy, but m3b.4 stores Cloud Run rows under `kind='compute.function'` so the existing `LookupServerlessFunction` path is reused without a new catalog lookup. The billing shape (per-request + per-vCPU-second + per-GB-second) is identical to Cloud Functions, so this does not mis-describe the price dimensions — only the taxonomy bucket is off by one. The debt is tracked in the plan's explicit non-goals; a future `schema_version` bump (post-v1) may re-key Cloud Run rows to `compute.container` once `sku compare --kind compute.container` becomes a real query target.

**Explicit non-goals for m3b.4** (declared up front to prevent scope creep):
- `gcp-bigquery`, `gcp-gke`, `gcp-cloud-cdn` → post-v1 / separate sub-plan.
- Cloud Run `--kind compute.container` reclassification → deferred per the kind-taxonomy note above.
- Cloud Functions 1st-gen vs 2nd-gen distinction → m3b.4 ingests 2nd-gen (gen2) SKUs only; gen1 deferred past v1.0 (Cloud Functions gen1 uses a distinct `resourceGroup`).
- Cloud Storage multi-region (`MULTI-REGION`) + dual-region SKUs — m3b.4 ships single-region rows only (fixtures filter `serviceRegions` to single-region entries); multi-/dual-region arrives post-v1.
- Cloud Storage archive retrieval / early-deletion fees, class-A/B request SKUs split → deferred; m3b.4 emits `storage` + `read-ops` + `write-ops` dimensions only, matching the `azure-blob` shape.
- GCP validation harness (cross-check vs live Cloud Billing Catalog API via OIDC) — tracked for m3b.5 when m3a + m3b all share one validation runner.
- Daily CI pipeline (`data-daily.yml`) matrix extension for GCP shards — tracked for m3b.5.
- Live multi-page JSON paging (`nextPageToken`) in the real Catalog API — m3b.4 ingests one fixture file and treats `nextPageToken` as informational; live paging arrives with the daily CI workflow.
- Committed-Use Discounts (`gcp-gce-cud`, `gcp-cloud-sql-cud`) and any `*-commitments` shard.
- `sku search --kind storage.object --provider gcp` / `sku compare` cross-provider VM rows — already live from m4.1-search; m3b.4 only verifies the new rows *appear* in search results, not that compare produces correct equivalence (that is M4 proper).
- `internal/updater` delta chain / manifest walking / ETag — remains a one-shot baseline download until m4+ per §9.

**Out-of-scope per shard:**
- `gcp-gcs`: MULTI-REGION / DUAL-REGION SKUs, Object Lifecycle transition SKUs, replication SKUs, egress / network SKUs (those are a separate `gcp-network` shard future), early-delete fees, retrieval fees.
- `gcp-run`: 1st-gen Cloud Run (legacy) SKUs, CPU-always-allocated (`CPU Allocation Time` second-gen only), custom-CPU / GPU-backed SKUs, Direct VPC Egress SKUs.
- `gcp-functions`: 1st-gen Cloud Functions SKUs, secret-access SKUs, networking egress SKUs.

# pipeline/ingest/gcp_gcs.py
"""Normalize GCP Cloud Billing Catalog JSON for Cloud Storage into sku row dicts.

Spec §5 kind=storage.object. One row per (storage_class, region) with three price
dimensions: `storage` (gb-mo), `read-ops` (requests), `write-ops` (requests).

resourceGroup -> canonical storage-class slug (StandardStorage -> standard).
meter description keyword -> dimension slug (Class A Operations -> read-ops,
Class B Operations -> write-ops, else storage).

Filters applied:
  - usageType == 'OnDemand'
  - currencyCode == 'USD'
  - region in region-normalizer (single-region only; multi/dual dropped)
  - resourceGroup in the canonical class set (drops MultiRegionalStorage,
    DualRegionalStorage, and any future class not yet mapped).
"""

from __future__ import annotations

import argparse
import sys
from pathlib import Path
from typing import Any, Iterable

from normalize.enums import apply_kind_defaults
from normalize.terms import terms_hash

from ._duckdb import dumps, open_conn
from .gcp_common import load_region_normalizer, parse_unit_price, parse_usage_unit

_PROVIDER = "gcp"
_SERVICE = "gcs"
_KIND = "storage.object"

_CLASS_MAP: dict[str, str] = {
    "StandardStorage": "standard",
    "NearlineStorage": "nearline",
    "ColdlineStorage": "coldline",
    "ArchiveStorage": "archive",
}

_CLASS_ATTRS: dict[str, dict[str, Any]] = {
    "standard": {"durability_nines": 11, "availability_tier": "standard"},
    "nearline": {"durability_nines": 11, "availability_tier": "infrequent"},
    "coldline": {"durability_nines": 11, "availability_tier": "cold"},
    "archive":  {"durability_nines": 11, "availability_tier": "archive"},
}

_SQL = """
WITH items AS (
  SELECT UNNEST(skus, recursive := true)
  FROM read_json_auto('{path}', maximum_object_size=33554432)
)
SELECT
  skuId                             AS sku_id,
  description                       AS description,
  category.resourceGroup            AS resource_group,
  category.usageType                AS usage_type,
  category.serviceDisplayName       AS service_display,
  serviceRegions                    AS regions,
  pricingInfo[1].pricingExpression.usageUnit                              AS usage_unit,
  pricingInfo[1].pricingExpression.tieredRates[1].unitPrice.units         AS price_units,
  pricingInfo[1].pricingExpression.tieredRates[1].unitPrice.nanos         AS price_nanos,
  pricingInfo[1].pricingExpression.tieredRates[1].unitPrice.currencyCode  AS currency
FROM items
"""


def _dimension(description: str) -> str:
    d = description.lower()
    if "class a operations" in d:
        return "read-ops"
    if "class b operations" in d:
        return "write-ops"
    return "storage"


def ingest(*, skus_path: Path) -> Iterable[dict[str, Any]]:
    norm = load_region_normalizer()
    conn = open_conn()
    cur = conn.execute(_SQL.format(path=str(skus_path).replace("'", "''")))
    raw: list[dict[str, Any]] = []
    for (sku_id, description, resource_group, usage_type, service_display,
         regions, usage_unit, price_units, price_nanos, currency) in cur.fetchall():
        if service_display != "Cloud Storage" or usage_type != "OnDemand":
            continue
        if currency != "USD":
            continue
        if resource_group not in _CLASS_MAP:
            continue  # drops MultiRegional / DualRegional / future classes.
        raw.append({
            "sku_id": sku_id,
            "description": description,
            "storage_class": _CLASS_MAP[resource_group],
            "regions": list(regions) if regions else [],
            "usage_unit": usage_unit,
            "price_units": price_units,
            "price_nanos": price_nanos or 0,
        })

    # group by (class, region) → three dimensions
    grouped: dict[tuple[str, str], dict[str, Any]] = {}
    for entry in raw:
        for region in entry["regions"]:
            try:
                region_normalized = norm(_PROVIDER, region)
            except KeyError as exc:
                raise ValueError(f"unknown region: {region}") from exc
            key = (entry["storage_class"], region)
            bucket = grouped.setdefault(key, {"sku_id": entry["sku_id"], "region_normalized": region_normalized, "prices": {}})
            divisor, unit = parse_usage_unit(entry["usage_unit"])
            amount = parse_unit_price(units=entry["price_units"], nanos=entry["price_nanos"]) / divisor
            dim = _dimension(entry["description"])
            bucket["prices"][dim] = {"dimension": dim, "tier": "", "amount": amount, "unit": unit}

    # emit one row per (class, region) iff all three dimensions present
    for (storage_class, region), bucket in grouped.items():
        if set(bucket["prices"].keys()) != {"storage", "read-ops", "write-ops"}:
            continue
        row = {
            "sku_id": bucket["sku_id"],
            "provider": _PROVIDER,
            "service": _SERVICE,
            "kind": _KIND,
            "resource_name": storage_class,
            "region": region,
            "region_normalized": bucket["region_normalized"],
            "prices": [bucket["prices"][d] for d in ("storage", "read-ops", "write-ops")],
            "attrs": dict(_CLASS_ATTRS[storage_class]),
        }
        apply_kind_defaults(row, kind=_KIND)
        row["terms_hash"] = terms_hash(
            commitment=row["commitment"],
            tenancy=row["tenancy"],
            os=row["os"],
            support_tier=row["support_tier"],
            upfront=row["upfront"],
            payment_option=row["payment_option"],
        )
        yield row


def main() -> int:
    ap = argparse.ArgumentParser()
    ap.add_argument("--skus", required=True, type=Path)
    ap.add_argument("--out", required=True, type=Path)
    args = ap.parse_args()
    with args.out.open("w") as f:
        for row in ingest(skus_path=args.skus):
            f.write(dumps(row) + "\n")
    return 0


if __name__ == "__main__":
    sys.exit(main())
```

Spec §5 kind=compute.function (pragmatic; Cloud Run is technically
compute.container, but shares the serverless three-dimension billing shape
with Cloud Functions — see plan taxonomy note). One row per (region) with
dimensions cpu-second / memory-gb-second / requests, architecture=x86_64.

Filters: usageType=OnDemand, currency=USD, region in normalizer,
resourceGroup=CloudRunV2 (drops gen1 CloudRun rows).

from __future__ import annotations

import argparse
import sys
from pathlib import Path
from typing import Any, Iterable

from normalize.enums import apply_kind_defaults
from normalize.terms import terms_hash

from ._duckdb import dumps, open_conn
from .gcp_common import load_region_normalizer, parse_unit_price, parse_usage_unit

_PROVIDER = "gcp"
_SERVICE = "run"
_KIND = "compute.function"
_RESOURCE_NAME = "cloud-run"
_ARCHITECTURE = "x86_64"

_SQL = """
WITH items AS (
  SELECT UNNEST(skus, recursive := true)
  FROM read_json_auto('{path}', maximum_object_size=33554432)
)
SELECT
  skuId                             AS sku_id,
  description                       AS description,
  category.resourceGroup            AS resource_group,
  category.usageType                AS usage_type,
  category.serviceDisplayName       AS service_display,
  serviceRegions                    AS regions,
  pricingInfo[1].pricingExpression.usageUnit                              AS usage_unit,
  pricingInfo[1].pricingExpression.tieredRates[1].unitPrice.units         AS price_units,
  pricingInfo[1].pricingExpression.tieredRates[1].unitPrice.nanos         AS price_nanos,
  pricingInfo[1].pricingExpression.tieredRates[1].unitPrice.currencyCode  AS currency
FROM items
"""


def _dimension(description: str, usage_unit: str) -> str | None:
    d = description.lower()
    if usage_unit == "count" or "request" in d:
        return "requests"
    if "cpu" in d:
        return "cpu-second"
    if "memory" in d:
        return "memory-gb-second"
    return None


def ingest(*, skus_path: Path) -> Iterable[dict[str, Any]]:
    norm = load_region_normalizer()
    conn = open_conn()
    cur = conn.execute(_SQL.format(path=str(skus_path).replace("'", "''")))
    grouped: dict[tuple[str, str], dict[str, Any]] = {}
    for (sku_id, description, resource_group, usage_type, service_display,
         regions, usage_unit, price_units, price_nanos, currency) in cur.fetchall():
        if service_display != "Cloud Run" or usage_type != "OnDemand":
            continue
        if currency != "USD":
            continue
        if resource_group != "CloudRunV2":
            continue
        dim = _dimension(description, usage_unit)
        if dim is None:
            continue
        for region in (regions or []):
            try:
                region_normalized = norm(_PROVIDER, region)
            except KeyError as exc:
                raise ValueError(f"unknown region: {region}") from exc
            key = (_RESOURCE_NAME, region)
            bucket = grouped.setdefault(key, {"sku_id": sku_id, "region_normalized": region_normalized, "prices": {}})
            divisor, unit = parse_usage_unit(usage_unit)
            amount = parse_unit_price(units=price_units, nanos=price_nanos or 0) / divisor
            # Canonicalize unit names for requests dim ('requests' stays 'requests';
            # 's' stays 's'; 'GiBy.s' becomes 'gb-s'.)
            bucket["prices"][dim] = {"dimension": dim, "tier": "", "amount": amount, "unit": unit}

    for (resource_name, region), bucket in grouped.items():
        if set(bucket["prices"].keys()) != {"cpu-second", "memory-gb-second", "requests"}:
            continue
        row = {
            "sku_id": bucket["sku_id"],
            "provider": _PROVIDER,
            "service": _SERVICE,
            "kind": _KIND,
            "resource_name": resource_name,
            "region": region,
            "region_normalized": bucket["region_normalized"],
            "prices": [bucket["prices"][d] for d in ("cpu-second", "memory-gb-second", "requests")],
            "attrs": {"architecture": _ARCHITECTURE},
        }
        apply_kind_defaults(row, kind=_KIND)
        row["terms_hash"] = terms_hash(
            commitment=row["commitment"],
            tenancy=row["tenancy"],
            os=row["os"],
            support_tier=row["support_tier"],
            upfront=row["upfront"],
            payment_option=row["payment_option"],
        )
        yield row


def main() -> int:
    ap = argparse.ArgumentParser()
    ap.add_argument("--skus", required=True, type=Path)
    ap.add_argument("--out", required=True, type=Path)
    args = ap.parse_args()
    with args.out.open("w") as f:
        for row in ingest(skus_path=args.skus):

    return 0

    sys.exit(main())

# pipeline/ingest/gcp_functions.py
"""Normalize GCP Cloud Billing Catalog JSON for Cloud Functions (gen2) into sku row dicts.

Spec §5 kind=compute.function. One row per region, dimensions cpu-second /
memory-gb-second / requests, architecture=x86_64. gen1 rows (resourceGroup=
CloudFunctions) are filtered; only gen2 (CloudFunctionsV2) emits.
"""

from __future__ import annotations

import argparse
import sys
from pathlib import Path
from typing import Any, Iterable

from normalize.enums import apply_kind_defaults
from normalize.terms import terms_hash

from ._duckdb import dumps, open_conn
from .gcp_common import load_region_normalizer, parse_unit_price, parse_usage_unit

_PROVIDER = "gcp"
_SERVICE = "functions"
_KIND = "compute.function"
_RESOURCE_NAME = "cloud-functions"
_ARCHITECTURE = "x86_64"
_SERVICE_DISPLAY = "Cloud Functions"
_RESOURCE_GROUP_GEN2 = "CloudFunctionsV2"

_SQL = """
WITH items AS (
  SELECT UNNEST(skus, recursive := true)
  FROM read_json_auto('{path}', maximum_object_size=33554432)
)
SELECT
  skuId                             AS sku_id,
  description                       AS description,
  category.resourceGroup            AS resource_group,
  category.usageType                AS usage_type,
  category.serviceDisplayName       AS service_display,
  serviceRegions                    AS regions,
  pricingInfo[1].pricingExpression.usageUnit                              AS usage_unit,
  pricingInfo[1].pricingExpression.tieredRates[1].unitPrice.units         AS price_units,
  pricingInfo[1].pricingExpression.tieredRates[1].unitPrice.nanos         AS price_nanos,
  pricingInfo[1].pricingExpression.tieredRates[1].unitPrice.currencyCode  AS currency
FROM items
"""


def _dimension(description: str, usage_unit: str) -> str | None:
    d = description.lower()
    if usage_unit == "count" or "invocation" in d or "request" in d:
        return "requests"
    if "cpu" in d:
        return "cpu-second"
    if "memory" in d:
        return "memory-gb-second"
    return None


def ingest(*, skus_path: Path) -> Iterable[dict[str, Any]]:
    norm = load_region_normalizer()
    conn = open_conn()
    cur = conn.execute(_SQL.format(path=str(skus_path).replace("'", "''")))
    grouped: dict[tuple[str, str], dict[str, Any]] = {}
    for (sku_id, description, resource_group, usage_type, service_display,
         regions, usage_unit, price_units, price_nanos, currency) in cur.fetchall():
        if service_display != _SERVICE_DISPLAY or usage_type != "OnDemand":
            continue
        if currency != "USD":
            continue
        if resource_group != _RESOURCE_GROUP_GEN2:
            continue
        dim = _dimension(description, usage_unit)
        if dim is None:
            continue
        for region in (regions or []):
            try:
                region_normalized = norm(_PROVIDER, region)
            except KeyError as exc:
                raise ValueError(f"unknown region: {region}") from exc
            key = (_RESOURCE_NAME, region)
            bucket = grouped.setdefault(key, {"sku_id": sku_id, "region_normalized": region_normalized, "prices": {}})
            divisor, unit = parse_usage_unit(usage_unit)
            amount = parse_unit_price(units=price_units, nanos=price_nanos or 0) / divisor
            bucket["prices"][dim] = {"dimension": dim, "tier": "", "amount": amount, "unit": unit}

    for (resource_name, region), bucket in grouped.items():
        if set(bucket["prices"].keys()) != {"cpu-second", "memory-gb-second", "requests"}:
            continue
        row = {
            "sku_id": bucket["sku_id"],
            "provider": _PROVIDER,
            "service": _SERVICE,
            "kind": _KIND,
            "resource_name": resource_name,
            "region": region,
            "region_normalized": bucket["region_normalized"],
            "prices": [bucket["prices"][d] for d in ("cpu-second", "memory-gb-second", "requests")],
            "attrs": {"architecture": _ARCHITECTURE},
        }
        apply_kind_defaults(row, kind=_KIND)
        row["terms_hash"] = terms_hash(
            commitment=row["commitment"],
            tenancy=row["tenancy"],
            os=row["os"],
            support_tier=row["support_tier"],
            upfront=row["upfront"],
            payment_option=row["payment_option"],
        )
        yield row


def main() -> int:
    ap = argparse.ArgumentParser()
    ap.add_argument("--skus", required=True, type=Path)
    ap.add_argument("--out", required=True, type=Path)
    args = ap.parse_args()
    with args.out.open("w") as f:
        for row in ingest(skus_path=args.skus):
            f.write(dumps(row) + "\n")
    return 0


if __name__ == "__main__":
    sys.exit(main())
```

package sku

import (

func newGCPGCSCmd() *cobra.Command {
    c := &cobra.Command{Use: "gcs", Short: "Google Cloud Storage pricing"}
    c.AddCommand(newGCPGCSPriceCmd())
    c.AddCommand(newGCPGCSListCmd())
    return c
}

type gcpGCSFlags struct {
    storageClass string // standard | nearline | coldline | archive
    region       string
    commitment   string
}

func (f *gcpGCSFlags) bind(c *cobra.Command) {
    c.Flags().StringVar(&f.storageClass, "storage-class", "",
        "standard | nearline | coldline | archive (single-region only in m3b.4)")
    c.Flags().StringVar(&f.region, "region", "", "GCP region (e.g. us-east1)")
    c.Flags().StringVar(&f.commitment, "commitment", "on_demand", "on_demand (only on-demand shipped in m3b.4)")
}

func (f *gcpGCSFlags) terms() catalog.Terms {
    return catalog.Terms{Commitment: f.commitment}
}

func newGCPGCSPriceCmd() *cobra.Command {
    var f gcpGCSFlags
    c := &cobra.Command{
        Use:   "price",
        Short: "Price one GCP Cloud Storage class",
        RunE:  func(cmd *cobra.Command, _ []string) error { return runGCPGCS(cmd, &f, true) },
    }
    f.bind(c)
    return c
}

func newGCPGCSListCmd() *cobra.Command {
    var f gcpGCSFlags
    c := &cobra.Command{
        Use:   "list",
        Short: "List GCP Cloud Storage SKUs matching filters (region optional)",
        RunE:  func(cmd *cobra.Command, _ []string) error { return runGCPGCS(cmd, &f, false) },
    }
    f.bind(c)
    return c
}

func runGCPGCS(cmd *cobra.Command, f *gcpGCSFlags, requireRegion bool) error {
    s := globalSettings(cmd)
    if f.storageClass == "" {
        e := skuerrors.Validation("flag_invalid", "storage-class", "",
            "pass --storage-class <class>, e.g. --storage-class standard")
        skuerrors.Write(cmd.ErrOrStderr(), e)
        return e
    }
    if requireRegion && f.region == "" {
        e := skuerrors.Validation("flag_invalid", "region", "",
            "pass --region <gcp-region>")
        skuerrors.Write(cmd.ErrOrStderr(), e)
        return e
    }
    if s.DryRun {
        return output.EmitDryRun(cmd.OutOrStdout(), output.DryRunPlan{
            Command: "gcp gcs " + cmd.Use,
            ResolvedArgs: map[string]any{
                "storage_class": f.storageClass,
                "region":        f.region,
                "commitment":    f.commitment,
            },
            Shards: []string{shardGCPGCS},
            Preset: s.Preset,
        })
    }
    shardPath := catalog.ShardPath(shardGCPGCS)
    if _, err := os.Stat(shardPath); err != nil {
        e := shardMissingErr(shardGCPGCS)
        skuerrors.Write(cmd.ErrOrStderr(), e)
        return e
    }
    cat, err := catalog.Open(shardPath)
    if err != nil {
        e := &skuerrors.E{Code: skuerrors.CodeServer, Message: err.Error()}
        skuerrors.Write(cmd.ErrOrStderr(), e)
        return e
    }
    defer func() { _ = cat.Close() }()

    if stale := applyStaleGate(cmd, cat, shardGCPGCS, s); stale != nil {
        return stale
    }

    rows, err := cat.LookupStorageObject(context.Background(), catalog.StorageObjectFilter{
        Provider: "gcp", Service: "gcs",
        StorageClass: f.storageClass,
        Region:       f.region,
        Terms:        f.terms(),
    })
    if err != nil {
        wrapped := fmt.Errorf("gcp gcs %s: %w", cmd.Use, err)
        skuerrors.Write(cmd.ErrOrStderr(), wrapped)
        return wrapped
    }
    if len(rows) == 0 {
        e := skuerrors.NotFound("gcp", "gcs",
            map[string]any{"storage_class": f.storageClass, "region": f.region},
            "Try `sku schema gcp gcs` or drop --region for a list")
        skuerrors.Write(cmd.ErrOrStderr(), e)
        return e
    }
    return renderRows(cmd, rows, s)
}

package sku

import (
    "context"
    "fmt"
    "os"

    "github.com/spf13/cobra"

    "github.com/sofq/sku/internal/catalog"
    skuerrors "github.com/sofq/sku/internal/errors"
    "github.com/sofq/sku/internal/output"
)

const shardGCPFunctions = "gcp-functions"

func newGCPFunctionsCmd() *cobra.Command {
    c := &cobra.Command{Use: "functions", Short: "Google Cloud Functions (gen2) pricing"}
    c.AddCommand(newGCPFunctionsPriceCmd())
    c.AddCommand(newGCPFunctionsListCmd())
    return c
}

type gcpFunctionsFlags struct {
    architecture string
    region       string
    commitment   string
}

func (f *gcpFunctionsFlags) bind(c *cobra.Command) {
    c.Flags().StringVar(&f.architecture, "architecture", "x86_64", "x86_64 (only arch shipped in m3b.4)")
    c.Flags().StringVar(&f.region, "region", "", "GCP region (e.g. us-east1)")
    c.Flags().StringVar(&f.commitment, "commitment", "on_demand", "on_demand (only on-demand shipped in m3b.4)")
}

func (f *gcpFunctionsFlags) terms() catalog.Terms {
    return catalog.Terms{Commitment: f.commitment}
}

func newGCPFunctionsPriceCmd() *cobra.Command {
    var f gcpFunctionsFlags
    c := &cobra.Command{
        Use:   "price",
        Short: "Price Cloud Functions for one architecture + region",
        RunE:  func(cmd *cobra.Command, _ []string) error { return runGCPFunctions(cmd, &f, true) },
    }
    f.bind(c)
    return c
}

func newGCPFunctionsListCmd() *cobra.Command {
    var f gcpFunctionsFlags
    c := &cobra.Command{
        Use:   "list",
        Short: "List Cloud Functions SKUs matching filters (region optional)",
        RunE:  func(cmd *cobra.Command, _ []string) error { return runGCPFunctions(cmd, &f, false) },
    }
    f.bind(c)
    return c
}

func runGCPFunctions(cmd *cobra.Command, f *gcpFunctionsFlags, requireRegion bool) error {
    s := globalSettings(cmd)
    if f.architecture == "" {
        e := skuerrors.Validation("flag_invalid", "architecture", "", "pass --architecture x86_64")
        skuerrors.Write(cmd.ErrOrStderr(), e)
        return e
    }
    if requireRegion && f.region == "" {
        e := skuerrors.Validation("flag_invalid", "region", "", "pass --region <gcp-region>")
        skuerrors.Write(cmd.ErrOrStderr(), e)
        return e
    }
    if s.DryRun {
        return output.EmitDryRun(cmd.OutOrStdout(), output.DryRunPlan{
            Command: "gcp functions " + cmd.Use,
            ResolvedArgs: map[string]any{
                "architecture": f.architecture,
                "region":       f.region,
                "commitment":   f.commitment,
            },
            Shards: []string{shardGCPFunctions},
            Preset: s.Preset,
        })
    }
    shardPath := catalog.ShardPath(shardGCPFunctions)
    if _, err := os.Stat(shardPath); err != nil {
        e := shardMissingErr(shardGCPFunctions)
        skuerrors.Write(cmd.ErrOrStderr(), e)
        return e
    }
    cat, err := catalog.Open(shardPath)
    if err != nil {
        e := &skuerrors.E{Code: skuerrors.CodeServer, Message: err.Error()}
        skuerrors.Write(cmd.ErrOrStderr(), e)
        return e
    }
    defer func() { _ = cat.Close() }()

    if stale := applyStaleGate(cmd, cat, shardGCPFunctions, s); stale != nil {
        return stale
    }

    rows, err := cat.LookupServerlessFunction(context.Background(), catalog.ServerlessFunctionFilter{
        Provider: "gcp", Service: "functions",
        Architecture: f.architecture,
        Region:       f.region,
        Terms:        f.terms(),
    })
    if err != nil {
        wrapped := fmt.Errorf("gcp functions %s: %w", cmd.Use, err)
        skuerrors.Write(cmd.ErrOrStderr(), wrapped)
        return wrapped
    }
    if len(rows) == 0 {
        e := skuerrors.NotFound("gcp", "functions",
            map[string]any{"architecture": f.architecture, "region": f.region},
            "Try `sku schema gcp functions` or drop --region for a list")
        skuerrors.Write(cmd.ErrOrStderr(), e)
        return e
    }
    return renderRows(cmd, rows, s)
}
